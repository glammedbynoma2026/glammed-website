# Glammed by Noma — Website

Luxury makeup artistry website for Glammed by Noma, Hanoi, Vietnam.

## Files in this folder

| File | Purpose |
|------|---------|
| index.html | Complete website — all sections |
| icon-192.png | Small logo — browser tab, nav bar |
| icon-512.png | Large logo — hero section image |
| logo.png | Full resolution logo |
| manifest.json | Makes site installable on phones |
| sw.js | Service worker — offline support |
| sitemap.xml | Tells Google all your pages |
| robots.txt | Tells Google it can index the site |
| vercel.json | Vercel deployment configuration |
| _headers | Security and caching headers |
| _redirects | URL redirect rules |

## Live URLs

- Website: https://www.glammedbynoma.com
- Loyalty App: https://app.glammedbynoma.com
- Instagram: https://instagram.com/glammedbynoma

## To make changes

1. Edit index.html on your computer
2. Go to GitHub → glammed-website repo
3. Click index.html → pencil icon → paste updated code → Commit
4. Vercel redeploys automatically in ~1 minute
